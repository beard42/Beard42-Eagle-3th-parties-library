This library consists of the following parts:

    Z80CPU-P -- CMOS Z80 CPU, DIP-40
    Z80CPU-V -- Z80 CPU, PLCC-44 SMD
    Z80CPU-S -- Z80 CPU, PLCC-44 Socket
    Z80CPU-F -- Z80 CPU, QFP-44 SMD
    Z80DMA-P -- Z80 DMA, DIP-40
    Z80DMA-V -- Z80 DMA, PLCC-44 SMD
    Z80DMA-S -- Z80 DMA, PLCC-44 Socket
    Z80DMANV -- NMOS Z80 DMA, PLCC-44 SMD
    Z80DMANS -- NMOS Z80 DMA, NMOS, PLCC-44 Socket
    Z80PIO-P -- Z80 PIO, DIP-40
    Z80PIO-V -- Z80 PIO, PLCC-44 SMD
    Z80PIO-S -- Z80 PIO, PLCC-44 Socket
    Z80PIO-F -- Z80 PIO, QFP-44 SMD
    Z80CTC-P -- CMOS Z80 CTC, DIP-28
    Z80CTC-V -- CMOS Z80 CTC, PLCC-44 SMD
    Z80CTC-S -- CMOS Z80 CTC, PLCC-44 Socket
    Z80CTC-F -- CMOS Z80 CTC, QFP-44 SMD
    Z80SIO-0 -- CMOS Z80 SIO/0, DIP-40
    Z80SIO-1 -- CMOS Z80 SIO/1, DIP-40
    Z80SIO-2 -- CMOS Z80 SIO/2, DIP-40
    Z80SIO-3 -- CMOS Z80 SIO/3, QFP-44 SMD
    Z80SIO-4 -- CMOS Z80 SIO/4, PLCC-44 SMD
    Z80SIO4S -- CMOS Z80 SIO/4, PLCC-44 Socket

The library is based on information contained in "Z80 Family Databook,"
Zilog literature number Q3/95 DB95Z800101.

Files (in addition to this README.TXT) are:

  Z80FAMIL.LBR -- the EAGLE library, Z80 Family
  LBRDIREC.TXT -- exported library directory
  LBR-SRC.ZIP -- library source code; EAGLE script files and
                 instructions to rebuild the library.
                 (use 'pkunzip -d' if you want subdirectories)
